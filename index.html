Blowfish-SMS by s7ephen

Like "OTR" for your phone. Send/Receive blowfish encrypted SMS messages.

Blowfish_sms is a script that runs on Android's SL4A (Scripting Layer For Android). It uses native API's to perform GUI and other "phoneish" things. It is intended to demonstrate the ease and power of having scripting languages available on your phone.

Dependencies

SL4A (Scripting Layer For Android) download it on your computer here, or use the phone on your camera to auto-download it here. Python For Android download it on your computer here, or use the phone on your camera to auto-download it here. Thats it! When I get the time to implement Blowfish in pure Java I may port this over a native Dalvik runtime (.apk). But for now I think the "openness" and readability of scripts makes it easier to "trust" an application of this kind, which is why this is a perfect Proof-of-Concept for SL4A.

Install

Install Scripting Layer For Android Install Python for Android Mount your phone as a USB Drive Download blowfish_sms.py and copy it to : sl4a/scripts on your phone.
